# Lar Certo 🐾

O **Lar Certo** é uma plataforma web desenvolvida para facilitar a adoção de animais, conectar ONGs e pessoas dispostas a dar um lar temporário ou definitivo para pets, além de ajudar a encontrar animais perdidos.

## 🎯 Objetivo
O principal objetivo do projeto é promover o bem-estar animal através da tecnologia, criando uma ponte entre animais que precisam de ajuda e pessoas dispostas a oferecer amor, cuidado ou recursos.

## ✨ Funcionalidades
A plataforma oferece diversas funcionalidades para os usuários:
- **Quero Adotar**: Visualização de pets disponíveis para adoção, com filtros por cidade.
- **Quero Doar/Cadastrar Pet**: Área para cadastrar animais que precisam de um novo lar.
- **Lares de Adoção**: Espaço para lares temporários e definitivos.
- **Perdi um Pet**: Seção dedicada para reportar e procurar animais desaparecidos.
- **Blog**: Artigos e dicas sobre o mundo pet.
- **ONGs**: Informações e contato de Organizações Não Governamentais parceiras.
- **Acessibilidade**: Integração com a ferramenta VLibras, garantindo que usuários com deficiência auditiva possam interagir com a plataforma.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as principais tecnologias base do desenvolvimento web front-end:
- **HTML5**: Semântico e estruturado.
- **CSS3**: Estilização responsiva e componentizada.
- **JavaScript (Vanilla)**: Interatividade, sliders e validações.

## 📁 Estrutura do Projeto
A estrutura de pastas do projeto é dividida da seguinte forma:
- `css/`: Contém os arquivos de folha de estilo da aplicação (ex: `style.css`).
- `js/`: Contém os scripts utilizados para lógica de front-end.
- `img/`: Imagens e assets utilizados no site (banners, logos, imagens de pets).
- `html`: Páginas principais da plataforma (`inicio.html`, `adotar.html`, `login.html`, etc.).
